mdTodo

iCloud上のMarkdownファイルでTodoを管理するiOSアプリ

アプリ概要

mdTodoは、iCloud Drive上のMarkdownファイルを直接編集してタスク管理を行うiPhoneアプリです。シンプルな操作でTodoの作成・完了・整理ができます。

Obsidianなど他のMarkdownエディタと同じファイルを共有できるため、iPhoneでもMacでも同じTodoリストにアクセスできます。

主な機能

  • iCloud Drive上の .md ファイルを直接読み書き
  • Markdown標準のタスクリスト形式 (- [ ] / - [x])
  • Obsidian互換 - 同じVaultのファイルをそのまま利用可能
  • ホーム画面ウィジェット対応（小・中・大）
  • Apple Watch コンプリケーション対応
  • アカウント登録不要・オフラインで動作
